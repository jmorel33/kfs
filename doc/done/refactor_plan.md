```
│   └── obj/ lib/ bin/         # Generated by make
├── scripts/

├── tests/                     # Harness sources (see doc/test_harness_plan.md)
├── doc/
│   ├── architecture.md        # Internal design deep dive
│   ├── COMPILATION_GUIDE.md   # Build and link reference
│   ├── test_harness_plan.md   # Harness phases H0–H7

├── build_kfs_smoke.bat        # Thin wrapper → build/build_smoke.bat
└── lib_kfs.h                  # Deprecated shim → kfs/kfs/kfs.h
```

### Include contract

```c
// Add -I sit/kfs to compile flags, then:

// In exactly one translation unit:
#define KFS_IMPLEMENTATION
#include "kfs/kfs.h"

// All other files:
#include "kfs/kfs.h"
```

**Breaking change vs old monolith:** the old `lib_kfs.h` always pulled implementation after `#endif`. The new layout requires `KFS_IMPLEMENTATION` in one TU (MyBuddy pattern).
